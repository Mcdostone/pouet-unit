# Web application

In order to have a visual render of the library, I want to create a web application that allows users to input values and select units for conversion. The application will display the converted results in real-time.

the web application will use the WASM export of the `pouet_jane` library, allowing for efficient and fast conversions directly in the browser.
The web application codebase will be located in a frontend directory. I must use modern vanilla JavaScript, HTML, and CSS to create a user-friendly interface for the unit conversion features provided by the `pouet_jane` library. Tailwindcss can be used for styling to ensure a responsive and visually appealing design. The application will include input fields for the value to be converted, dropdowns for selecting the source and target units, and a display area for showing the conversion results.

The web application must use the latest version of viteJS.
